# 原子操作

```php
if($fp=fopen($datafile,'r+')){
	//开始时间
    $startTime=microtime();
    do{
   		//取得文件的独占权限（锁文件）
        $canWrite=flock($fp,LOCK_EX);
        if(!$canWrite){
  			//没权限则其他程序在执行，等待
            usleep(round(rand(0,100)*1000));
        }
		//不能写的时候继续执行  （考虑到排队时间）
    }while((!$canWrite)&&((microtime()-$startTime)<1000));
    //可以写了，执行，然后文件解锁
    if($canWrite){
    	//TODO something
        //释放 文件锁
        @flock($fp, LOCK_UN);
    }
    fclose($fp);
}
```
